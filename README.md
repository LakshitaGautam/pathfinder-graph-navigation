# PathFinder: Graph-Based Navigation System

A graph traversal and shortest-path system implementing BFS, DFS, and
Dijkstra's Algorithm from scratch in C++.

Built as part of Data Structures & Algorithms coursework at Jaypee
Institute of Information Technology, Noida (B.Tech CSE, 2023-28).

## Tech Stack
C++

## Algorithms Implemented
- BFS (Breadth-First Search) — unweighted shortest path using Queue
- DFS (Depth-First Search) — full traversal using Stack
- Dijkstra's Algorithm — weighted shortest path using Priority Queue

## Graph Representations
- Adjacency Matrix
- Adjacency List

## How to Run
    g++ pathfinder.cpp -o pathfinder
    ./pathfinder
